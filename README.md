#### What is the purpose of this mod?
In the base game, the only nation which can create puppets from decisions is Germany with its Reichskommissariats. This mod attempts to address this, by adding new formable puppets which nations can create if they occupy certain states. The newly added puppets have a custom autonomy level named "Military Occupation" which mirrors the effects of the base game "Integrated Puppet".
#### Current Features:
- Ability to release up to 123 puppets through new decisions which are highlighted on the map.
- Addition of a new autonomy level, "Military Occupation", which matches the effects of an Integrated Puppet.
- Addition of a puppet management user interface which allows a player to view the states and cores of a puppet, appoint a military governor, and dismiss the current military governor.
- Ability to appoint army and navy leaders as military governors (works for all nations, and leaders added through mods).
- Puppet's ideology will change automatically to match the overlord's ideology.
- Ability to hand back territory through a state transfer user interface to create your desired borders.
- Optional resistance system — when enabled, newly formed puppets contest their states instead of immediately coring them.
- Ability to send manpower and equipment directly to struggling puppets to keep them afloat.
- Overlord color option — display all your puppets in your nation's color.
#### Current Compatibility?
As this modification targets vanilla state IDs, any other mod that adds/modifies states will be incompatible (ex. Beautiful States, Satisfying States).\
